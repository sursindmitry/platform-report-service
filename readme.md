# Platform Report Service

## How to start?
Add `.env` file: 
````
# Kafka
SERVER_HOST_KAFKA= YOUR KAFKA SERVER
SERVER_PORT_KAFKA= YOUR KAFKA PORT

# Mongo
SERVER_MONGO_USERNAME= YOUR MONGO USERNAME
SERVER_MONGO_PASSWORD= YOUR MONGO PASSWORD
SERVER_HOST_MONGO= YOUR MONGO SERVER
SERVER_PORT_MONGO= YOUR MONGO PORT

# Redis
SERVER_HOST_REDIS= YOUR REDIS SERVER
SERVER_PORT_REDIS= YOUR REDIS PORT

# Postgres
SERVER_POSTGRES_HOST= YOUR POSTGRES SERVER
SERVER_POSTGRES_PORT= YOUR POSTGRES PORT
SERVER_POSTGRES_DB= YOUR POSTGRES DB NAME
SERVER_POSTGRES_USERNAME= YOUR POSTGRES USERNAME
SERVER_POSTGRES_PASSWORD= YOUR POSTGRES PASSWORD
````
